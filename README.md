# Keewatin Self Storage — Website

## Open in VS Code
```
File → Open Folder → select this folder (keewatin-website)
```

## Files
- `index.html` — the complete website (single file, ready to deploy)
- `images/` — drop any custom photos here and update the `src` paths in index.html

## To Deploy
1. Upload `index.html` to your web host and rename it or set it as the homepage
2. Or paste the contents into your WordPress custom HTML template
